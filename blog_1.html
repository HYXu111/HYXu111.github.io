Error: "No pyface.toolkits plugin could be loaded for qt4" on Debian

Problem: This error occurs when running applications requiring Qt4 on Debian 12+.

Solution: Set these environment variables:

export ETS_TOOLKIT=qt

export QT_API=pyqt5

Note: Add to ~/.bashrc for persistence.
